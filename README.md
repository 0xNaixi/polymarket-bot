## Features

- **Account Registration** : Easily register new Polymarket accounts.
- **Proxy Support** : Send requests via proxies to enhance security and anonymity.
- **Efficient Processing** : Generate organized output files for streamlined account management.

## Prerequisites

Before you begin, ensure you have met the following requirements:

- **Rust** : Install Rust by following the instructions on [rust-lang.org]().
- **Git** : Ensure Git is installed on your machine. Download it from [git-scm.com](https://git-scm.com/downloads) if necessary.
